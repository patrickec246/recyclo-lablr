Option: Load Available Polygons

As mentioned before, your results are averaged with other labelers' results to improve consistency.

If 'Load Polygons From Server' is selected, when you receive a new image to label, we'll also provide you with the average polygons other labelers have submitted. These can help speed up labeling since these already-labeled polygons usually only require verification or minor tweaking instead of manually adding all new polygons.
